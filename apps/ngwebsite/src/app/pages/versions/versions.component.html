Angular versions

A list of current Angular versions and the related features. View the Changelog to show all updates.

Angular 8

  • Ivy
  • Typescript 3.4
  • Bazel support
  • Forms: markAllAsTouched() and FormArray.clear
  • Router: Lazy-loading with import() syntax, PlatformLocation

Angular 7

Here is the announcement on the official Angular blog.

  • CLI Prompts
  • Application Performance: reflect-metadata polyfill included.
  • Angular Material & the CDK: virtual scrolling, drag and drop
  • Improved Accessibility of Selects
  • Angular Elements

Angular 6

Amazing features has been added with this version.

  • ng add, add package
  • ng update, update and upgrade packages
  • Declaring the providers inside the service itself
  • Use ng-template instead of template directive
  • Angular Elements
  • Upgrading to RxJS 6.0.0

Angular 5

  • Http (@angular/http) deprecated, HttpClient (@angular/common/http) is now the new library
  • Multiple Alias export
  • Internationalized Number, Date, and Currency Pipes
  • Improved Decorator Support: expression lowering in decorators for lambdas, and the value of useValue, useFactory, and data in object literals.
  • Build Optimization
  • Angular Universal Transfer API: Domino added to the platform server. ServerTransferStateModule and BrowserTransferModule have been added to Angular Universal.
  • Faster compiler with ng serve --aot
  • Form validation
  • Animations: new transition aliases, :increment and :decrement
  • Better Support for Service Workers

Angular 4

The version 3 was skipped.

  • Ahead of Time compilation - View Engine
  • Angular Universal
  • Angular Animations
  • ng-template replaces the deprecated template
  • NgIf with else
  • Titlecase Pipe

Angular 2

Maintained by Google, based on NodeJS and written in Typescript, Angular 2 was a great innovation: a new and dynamic website development framework was born.
